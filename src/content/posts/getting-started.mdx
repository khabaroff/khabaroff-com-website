# Начало работы

Этот пост покажет вам, как работать с Tina CMS.

## Запуск редактора

Для редактирования контента:

1. Запустите `npm run dev` - Astro dev сервер
2. Запустите `npx tinacms dev` - Tina CMS сервер
3. Откройте `/admin` в браузере

## Редактирование постов

В Tina admin панели вы можете:

- Создавать новые посты
- Редактировать существующие
- Добавлять кастомные блоки контента
- Видеть превью изменений в реальном времени

## Сохранение

Все изменения сохраняются напрямую в `.mdx` файлы в вашем репозитории.
