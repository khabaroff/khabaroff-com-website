# GTD: система управления задачами, которая работает

Getting Things Done (GTD) — это не просто to-do список. Это целая система освобождения мозга от необходимости помнить всё.

## Суть методологии

<Callout type="info">
**Главный принцип:** Мозг создан для обработки информации, а не для её хранения.
</Callout>

Ключевая идея GTD — выгрузить ВСЁ из головы во внешнюю систему.

## Пять шагов GTD


### 3. Организация (Organize)

Раскладывайте по категориям:
- **Next Actions** — следующие действия
- **Projects** — проекты (>1 действия)
- **Waiting For** — ждете от других
- **Someday/Maybe** — когда-нибудь

### 4. Рефлексия (Reflect)

**Ежедневный обзор:**
- Просмотр календаря
- Проверка Next Actions
- Обновление списка Waiting For

**Еженедельный обзор:**
- Обработка входящих
- Обзор проектов
- Планирование недели

### 5. Действие (Engage)

Выбирайте задачи по контексту:
- @компьютер
- @дом
- @звонки
- @поручения

<Callout type="success">
**Фишка:** Группируйте похожие задачи и делайте пачками — так эффективнее.
</Callout>

## Моя адаптация GTD для 2026

**Цифровые инструменты:**
- **Todoist** — основная система
- **Notion** — базы знаний и проекты
- **Apple Notes** — быстрый сбор

**Упрощения:**
- Меньше контекстов (только 3-4 основных)
- Еженедельный обзор вместо ежедневного
- AI помогает категоризировать задачи

## Типичные ошибки

<Callout type="warning">
**Не делайте так:**
- Перфекционизм в настройке системы
- Слишком сложная структура
- Сбор без обработки
- Пропуск еженедельных обзоров
</Callout>

## Минимальная версия GTD

Если вся система кажется сложной, начните с этого:

1. Один inbox для всех входящих
2. Обрабатывайте inbox раз в день
3. Ведите список Next Actions
4. Раз в неделю делайте обзор

Этого достаточно для 80% результата.

## Заключение

GTD работает, но требует дисциплины. Первые 2-3 недели будет непривычно, потом система начнет окупаться свободным вниманием и спокойствием.

**Главное:** Начните с малого. Полная система GTD — это марафон, а не спринт.
